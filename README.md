# 🐧 Colección de Scripts en Bash

Este repositorio contiene una colección de scripts desarrollados en **Bash** con el objetivo de automatizar tareas comunes en sistemas GNU/Linux.

## 📌 Descripción

Los scripts incluidos están diseñados para facilitar tareas de administración, automatización y uso cotidiano del sistema. Cada script es independiente y puede utilizarse según la necesidad.

## ⚙️ Requisitos

- Sistema operativo GNU/Linux
- Intérprete Bash (generalmente ya instalado)
- Dependencias específicas (indicadas en cada script si corresponde)

## 🚀 Uso

1. Clonar el repositorio:

```bash
git clone https://github.com/due204/bash-script.git
cd bash-script
```

2. Dar permisos de ejecución al script:

```bash
chmod +x nombre_script.sh
```

3. Ejecutar el script:

```bash
./nombre_script.sh
```
